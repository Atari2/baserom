Baserom Levels

Info Level

The baserom makes use of level 105 to display a bunch of information about the baserom so it's seen right when you open Lunar Magic. This level is normally Yoshi's Island 1 but that has been moved to the unused level 112.

Demo Level

A demo level that is designed to present the some of custom resources is included in the baserom, and is accessible both in-ROM at level 13B and from the Overworld by going up from Yoshi's House so you can try it for yourself.

Resource Credits

It is good practice to keep track of all resources used in your hacks, if you can help it, too give credit to their authors when working on a project, that way you can include them in a credits file when you go to release your hack.

See the included "CREDITS.txt" file in this baserom as an example. This also provided so you can see all resources included.
